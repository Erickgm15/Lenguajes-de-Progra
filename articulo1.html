HTML

HTML, siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar que se utiliza para crear páginas web. Es la estructura básica de cualquier sitio en Internet y permite definir elementos como textos, imágenes, enlaces, formularios, tablas y mucho más.

¿Cómo funciona HTML?

HTML funciona mediante etiquetas o "tags" que le indican al navegador cómo debe mostrar el contenido. Estas etiquetas están rodeadas por signos de menor y mayor que (por ejemplo, para un párrafo). Cada elemento generalmente tiene una etiqueta de apertura y una de cierre.

Estructura básica de un documento HTML

  • DOCTYPE html indica que se está utilizando HTML5.
  • html contiene todo el documento.
  • incluye información como el título y la codificación del texto.
  • bosdy contiene el contenido visible de la página.

    • Principales etiquetas HTML

  • h1 a h6: Encabezados de mayor a menor tamaño.
  • p: Párrafos de texto.
  • a href="...": Enlaces o hipervínculos.
  • img src="...": Inserción de imágenes.
  • ul, ol, li: Listas desordenadas y ordenadas.
  • div y span: Contenedores para organizar el contenido.
